Também estudei o funcionamento do software, incluindo sistemas operacionais e sua interação com o hardware para gerenciar processos e recursos. DashCO - Plataforma Administrativa (Desenolvimento)

O DashCo é uma plataforma moderna e intuitiva desenvolvida para otimizar a produtividade e organização dos usuários. A aplicação conta com um dashboard interativo que centraliza diversas ferramentas essenciais, como um chat integrado com IA utilizando a API da OpenAI, um sistema de gerenciamento de tarefas que permite criar, editar e concluir atividades, além de um bloco de notas digital para anotações rápidas. Também inclui uma calculadora embutida, suporte para armazenamento e acesso a arquivos, uma agenda para eventos e compromissos, e um módulo que permite a edição de planilhas Excel diretamente na plataforma. Projetado com um design clean e responsivo, o DashCo oferece uma experiência fluida, garantindo facilidade de uso e eficiência. Desenvolvido com tecnologias modernas como HTML, CSS e JavaScript no frontend e Node.js no backend, a plataforma é hospedada no Render para acessibilidade global.

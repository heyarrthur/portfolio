Análise e Desenvolvimento de Sistemas

Atualmente estou cursando o 2º período de Análise e Desenvolvimento de Sistemas pela Rede de Ensino Doctum, onde venho desenvolvendo uma base sólida nos fundamentos da tecnologia, da programação e da lógica computacional, essenciais para a criação de sistemas e soluções digitais.

Até este momento da graduação, minha formação tem sido focada no desenvolvimento de competências fundamentais na área de desenvolvimento de software, banco de dados e infraestrutura de tecnologia.

🚀 Principais Conhecimentos e Habilidades Adquiridos até o 2º Período:

🔹 Lógica de Programação e Algoritmos: Desenvolvimento de raciocínio lógico aplicado à resolução de problemas, criação de algoritmos e estruturas de controle, fundamentais para qualquer desenvolvimento de software.

🔹 Introdução à Programação: Primeiros passos na construção de sistemas utilizando linguagens de programação estruturada, aprendendo conceitos como variáveis, condicionais, loops, funções e vetores.

🔹 Arquitetura e Organização de Computadores: Entendimento do funcionamento dos componentes de hardware, sistemas operacionais, processamento, armazenamento e comunicação entre dispositivos.

🔹 Banco de Dados (Fundamentos): Conhecimentos introdutórios em modelagem de dados, criação de tabelas, consultas, manipulação de dados e conceitos básicos de SQL, fundamentais para a gestão de dados em sistemas.

🔹 Redes de Computadores (Fundamentos): Primeiros conceitos sobre comunicação de dados, funcionamento de redes, protocolos, dispositivos de rede e noções iniciais sobre segurança e infraestrutura.

🔹 Sistemas de Informação: Entendimento sobre como os sistemas são aplicados no contexto organizacional, seus impactos nos processos de negócio e sua importância na tomada de decisões.

🔹 Empreendedorismo e Inovação: Desenvolvimento de uma visão empreendedora aplicada à tecnologia, com foco em criatividade, solução de problemas, identificação de oportunidades e inovação no mercado de TI.

📈 Resumo da Formação em Andamento

Ao longo desses dois primeiros períodos, tenho fortalecido meu conhecimento nas bases do desenvolvimento de sistemas, raciocínio lógico, banco de dados e fundamentos de redes e arquitetura de computadores. Essa etapa da minha formação tem sido fundamental para construir a estrutura técnica necessária para avançar em áreas mais complexas do desenvolvimento de software, análise de sistemas e soluções digitais.

Desenvolvimento Full Stack Java

Durante minha jornada no curso de Desenvolvimento Full Stack Java pela EBAC (Escola Britânica de Artes Criativas e Tecnologia), adquiri uma formação completa que me capacita a atuar em todas as etapas do desenvolvimento de aplicações web, tanto no front-end quanto no back-end.

O curso proporcionou uma experiência prática e aprofundada, onde pude desenvolver projetos reais, compreender os fluxos de desenvolvimento e aplicar as principais tecnologias e metodologias utilizadas no mercado de tecnologia.

🚀 Principais Competências e Conhecimentos Adquiridos:

🔸 Front-End (Interface do Usuário) Criação de páginas web estruturadas com HTML5 e estilizadas com CSS3, utilizando boas práticas de design responsivo.

Desenvolvimento de interatividade e dinamismo nas interfaces utilizando JavaScript puro e bibliotecas como jQuery.

Aplicação de frameworks e pré-processadores como Bootstrap, Flexbox e SASS, garantindo layouts modernos e profissionais.

Integração com serviços externos e APIs, utilizando AJAX para comunicação assíncrona e atualização dinâmica dos dados.

Controle de versão e colaboração em projetos com Git e GitHub, aplicando fluxo de trabalho profissional.

🔸 Back-End (Lógica de Negócio e Servidores)

Domínio da linguagem Java, com foco na programação orientada a objetos e boas práticas de desenvolvimento.

Criação de APIs robustas e seguras utilizando o framework Spring Boot, além de implementação de microsserviços e gerenciamento de dependências com Maven e Gradle.

Desenvolvimento de sistemas back-end capazes de processar, armazenar e gerenciar dados de forma eficiente.

Criação e gestão de bancos de dados relacionais, utilizando SQL, além de realizar consultas, manipulações e modelagem de dados.

Aplicação de testes unitários e de integração, garantindo a qualidade, segurança e estabilidade das aplicações.

🔸 DevOps e Deploy

Configuração de ambientes de desenvolvimento modernos.

Uso de ferramentas como Docker para containerização, facilitando o deploy e a escalabilidade de aplicações.

Conhecimentos introdutórios sobre integração contínua e entrega contínua (CI/CD), aplicando práticas de DevOps no ciclo de desenvolvimento.

🎯 Resumo da Formação

Ao longo dessa formação, desenvolvi habilidades técnicas e práticas essenciais para atuar como desenvolvedor Full Stack. Hoje sou capaz de construir soluções web completas, desde a criação da interface do usuário até a lógica de negócios no servidor, utilizando tecnologias modernas, seguras e escaláveis.

Essa experiência me proporcionou não apenas conhecimento técnico, mas também visão de mercado, capacidade de trabalho em equipe, versionamento de projetos e desenvolvimento de software com foco na qualidade e na entrega de valor.

Conhecimentos

HTML

Tempo de Experiência: 2.5 anos | Nível: Avançado

HTML (HyperText Markup Language) é a linguagem de marcação responsável por estruturar o conteúdo de páginas na web. Com ele, definimos elementos como títulos, parágrafos, imagens, links, tabelas e formulários, funcionando como o esqueleto que dá forma e organização ao site.

CSS

Tempo de Experiência: 2 anos | Nível: Avançado

CSS (Cascading Style Sheets) é a linguagem utilizada para definir o estilo e o design de páginas web. Com o CSS, é possível personalizar cores, tamanhos, fontes, espaçamentos, posicionamento de elementos e criar efeitos visuais, deixando o site mais bonito, organizado e responsivo.

SCSS

Tempo de Experiência: 1 ano | Nível: Básico

SCSS (Sassy CSS) é uma sintaxe do Sass, um pré-processador de CSS que permite escrever estilos de forma mais organizada, reutilizável e eficiente. Com SCSS, é possível usar variáveis, funções, aninhamento de seletores, mixins e outras funcionalidades que não existem no CSS puro, tornando o desenvolvimento mais rápido e produtivo.

JSON

Tempo de Experiência: 2 anos | Nível: Intermediário

JSON (JavaScript Object Notation) é um formato leve de troca de dados, fácil de ler e escrever tanto para humanos quanto para máquinas. Ele é usado principalmente para transmitir informações entre um servidor e uma aplicação web, utilizando uma estrutura de chave e valor. É amplamente utilizado em APIs e sistemas modernos para enviar e receber dados de forma organizada.

JavaScript

Tempo de Experiência: 2.5 anos | Nível: Avançado

JavaScript é uma linguagem de programação utilizada principalmente no Front-End para tornar as páginas web interativas e dinâmicas. Com JavaScript, é possível criar efeitos visuais, validar formulários, manipular elementos da página em tempo real, além de se comunicar com servidores para carregar dados sem precisar recarregar a página. Hoje, também é muito usado no Back-End, com ambientes como Node.js, para criar aplicações completas.

Boostrap

Tempo de Experiência: 2 anos | Nível: Intermediário

Bootstrap é um framework front-end gratuito e open source que facilita a criação de sites e aplicações web responsivas e modernas. Ele oferece um conjunto de estilos CSS prontos, componentes interativos e um sistema de grid flexível, permitindo que desenvolvedores construam layouts rápidos, consistentes e adaptáveis a diferentes tamanhos de tela, sem precisar criar tudo do zero.

VueJS

Tempo de Experiência: 1 ano | Nível: Básico

Vue.js é um framework progressivo de JavaScript para construir interfaces de usuário. Ele é focado na camada de visualização (View) e permite criar aplicações web reativas e componentes reutilizáveis de forma simples e eficiente. Com Vue, você pode construir desde pequenos widgets até aplicações complexas de página única (SPA), facilitando a organização e manutenção do código.

ReactJS

Tempo de Experiência: 1 ano | Nível: Intermediário

React.js é uma biblioteca JavaScript criada pelo Facebook para construir interfaces de usuário de forma eficiente e dinâmica. Ela permite criar componentes reutilizáveis que gerenciam seu próprio estado, facilitando o desenvolvimento de aplicações web interativas e de página única (SPA). Com React, o DOM é atualizado de forma rápida e otimizada graças ao uso do Virtual DOM.

NodeJS

Tempo de Experiência: 1 ano | Nível: Avançado

Node.js é um ambiente de execução JavaScript do lado do servidor que permite rodar código JavaScript fora do navegador. Ele é baseado no motor V8 do Chrome e é muito usado para construir aplicações de back-end rápidas, escaláveis e em tempo real, como APIs, servidores web e sistemas distribuídos. Além disso, permite usar JavaScript no servidor, unificando a linguagem no front-end e no back-end.

JAVA

Tempo de Experiência: 1.5 ano | Nível: Intermediário

Java é uma linguagem de programação orientada a objetos, amplamente usada para desenvolver aplicações de grande porte, sistemas corporativos, aplicativos móveis (Android), e sistemas embarcados. Conhecida por sua portabilidade, ela roda em qualquer plataforma que tenha a Java Virtual Machine (JVM), garantindo o lema "write once, run anywhere" (escreva uma vez, execute em qualquer lugar).

MongoDB

Tempo de Experiência: 1.5 ano | Nível: Avançado

MongoDB é um banco de dados NoSQL orientado a documentos, que armazena dados em formato JSON-like (BSON). Ele é flexível, escalável e ideal para aplicações que precisam lidar com grandes volumes de dados e estruturas dinâmicas, como dados sem esquema fixo. MongoDB é muito usado em sistemas modernos por sua facilidade de integração com linguagens como JavaScript e por permitir consultas rápidas e eficientes.

SpringBoot

Tempo de Experiência: 1 anos | Nível: Intermediário

Spring Boot é um framework Java que simplifica a criação de aplicações web e microserviços. Ele oferece uma configuração automática e pronta para uso, facilitando o desenvolvimento rápido e eficiente de aplicações robustas, com menos código e configuração manual. Com Spring Boot, é fácil integrar bancos de dados, segurança, APIs REST e outras funcionalidades essenciais para aplicações corporativas.

GIT

Tempo de Experiência: 2 anos | Nível: Intermediário

Git é um sistema de controle de versões distribuído usado para gerenciar e acompanhar mudanças no código-fonte de projetos. Ele permite que múltiplos desenvolvedores trabalhem simultaneamente, salvando diferentes versões do código, facilitando colaboração, histórico de alterações, ramificações (branches) e fusões (merges) de forma eficiente e segura. É fundamental no desenvolvimento moderno de software.

Figma

Tempo de Experiência: 1 ano | Nível: Intermediário

Figma é uma ferramenta de design de interface e prototipagem colaborativa baseada na web. Ela permite que equipes criem, compartilhem e editem designs em tempo real, facilitando o desenvolvimento de layouts para sites, aplicativos e sistemas, com recursos para criar protótipos interativos sem a necessidade de instalar nenhum software.

Docker

Tempo de Experiência: 1 ano | Nível: Básico

Docker é uma plataforma que permite criar, distribuir e executar aplicações dentro de contêineres — ambientes isolados que incluem tudo o que o software precisa para rodar, como código, bibliotecas e dependências. Isso garante que a aplicação funcione de forma consistente em qualquer lugar, facilitando a implantação, escalabilidade e o desenvolvimento colaborativo.

GitHub

Tempo de Experiência: 3 anos | Nível: Avançado

GitHub é uma plataforma online que hospeda repositórios Git, facilitando o controle de versões, colaboração e compartilhamento de código entre desenvolvedores. Além de armazenar o código-fonte, oferece ferramentas para revisão de código, gerenciamento de projetos, documentação e integração contínua, sendo muito popular na comunidade de desenvolvimento de software.

● 🔥 Sistema de Assinaturas: Oferece a possibilidade de criar planos de assinatura para clientes fiéis, gerando recorrência, fidelização e previsibilidade financeira para a barbearia.

● 🧠 Sistema de CRM: Gestão completa dos clientes, com histórico de serviços, preferências, frequência de visitas e informações que ajudam a criar um atendimento personalizado e de alto nível.

💡 Diferenciais do Sistema:

● Desenvolvido por barbeiros, para barbeiros, focando nas reais necessidades do mercado.

● Interface moderna, limpa e intuitiva, que oferece liberdade total na gestão do negócio, seja para pequenos salões ou grandes barbearias.

🧠 Tecnologias Utilizadas:

● HTML5 – Estruturação robusta e semântico do front-end.

● CSS3 – Design responsivo, moderno e elegante.

● JavaScript – Interatividade e funcionalidades dinâmicas no front-end e no back-end.

● Node.js – Backend rápido, escalável e eficiente, suportando grande volume de dados e múltiplos usuários simultâneos.

● Bootstrap – Framework de design para acelerar o desenvolvimento com responsividade nativa.

● MongoDB – Banco de dados NoSQL, perfeito para armazenar dados estruturados e não estruturados de clientes, agendas, serviços e muito mais.

A Thynk Barber se tornou uma plataforma referência para barbearias que buscam modernização, controle total do seu negócio e uma experiência digital tanto para os clientes quanto para os profissionais. DashCO

A DashCO foi o primeiro sistema que desenvolvi com foco em facilitar a rotina de trabalho e organização pessoal durante o dia a dia de serviços. Idealizada como uma central de produtividade, a DashCO reúne em um só lugar ferramentas úteis, intuitivas e voltadas para o uso local, garantindo segurança e praticidade.

O projeto foi desenvolvido com o apoio de tecnologias modernas e recursos de Inteligência Artificial, servindo como um ambiente pessoal onde o usuário pode se organizar com agilidade e segurança, sem depender da internet ou da nuvem.

⚙️ Funcionalidades Disponíveis:

● 🧮 Calculadora embutida: Rápida e prática para realizar cálculos durante qualquer tarefa.

● 🗒️ Bloco de notas: Para anotações rápidas, ideias, checklists e rascunhos.

● 🤖 Integração com o ChatGPT: Já funcional, permitindo interação com IA diretamente na interface. No entanto, requer uma estrutura de backend em Node.js rodando 24h para manter a funcionalidade ativa de forma contínua.

● 🌐 Sistema de adição de plataformas: Possibilidade de registrar e organizar links e sistemas utilizados no dia a dia, funcionando como um dashboard pessoal.

● ✅ Sistema de tarefas (To-Do): Permite criar, visualizar e gerenciar tarefas pessoais de forma simples.

🔒 Segurança e Armazenamento Local:

Todos os dados são armazenados via LocalStorage do navegador, o que significa que nada é enviado para servidores externos. Isso oferece um alto nível de privacidade e segurança, especialmente útil para usuários que lidam com informações sensíveis ou preferem evitar o uso de plataformas na nuvem.

🚧 Estado Atual e Futuras Atualizações:

O sistema ainda está em fase de melhorias. Embora funcional, a DashCO ainda não foi finalizada por completo, devido à falta de tempo para ajustes mais avançados. Entre as melhorias previstas estão:

● Backend dedicado para a IA funcionar 24h com segurança;

● Sistema de login local opcional;

● Personalização da interface;

● Exportação de dados;

● Integração com outras APIs.

A DashCO é um projeto pessoal que representa um marco inicial no meu desenvolvimento de sistemas, integrando funcionalidades reais com uma visão de segurança, utilidade e produtividade. Ela reflete minha vontade de criar soluções inteligentes, práticas e totalmente personalizadas para o dia a dia profissional.
